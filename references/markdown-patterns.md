# Markdown Patterns for rabbitQ-skill-lark-xhs

这份约定用于准备飞书导出的 Markdown。转换器会独立解析 Markdown，不需要先经过公众号 HTML。

## 封面标题与副标题

- **主标题**：优先 frontmatter `title`，否则第一个 `#` 标题，否则文件名。
- **副标题**：
  - 若 frontmatter 写了 `subtitle:`，或转换时传了 `--subtitle`，直接用；
  - **否则由执行转换的 AI 通读全文后自行总结**，通过 `--subtitle` 写入 Studio（`convert.js` 不会从正文自动抽取）。
  - 副标题宜一句话、约 48 字以内、最多两行，补充主标题而不重复。

`subtitle` 仍可在 frontmatter 里手写；与 AI 总结二选一即可，frontmatter 优先。

```markdown
---
title: "AI 视频一眼假怎么破？"
subtitle: "这次不是讲概念，是一次真实踩坑复盘。"
---
```

若省略 `subtitle`，交给 AI 在转换前读文档生成，例如：

```bash
node scripts/convert.js article.md --subtitle "一眼分清假视频问题，附可复用提示词思路"
```

## 标题

- 封面标题：文章第一个标题（通常是 H1）或 frontmatter `title`，不重复进正文。
- 正文标题按**文章实际出现的 `#` 深度动态映射**（排除封面标题后）：
  - 最少的那种 `#` → **一级标题**（自动编号 01/02 + 通栏分隔线）
  - 第二少的那种 `#` → **二级标题**（强调色字 + 2px 浅色下划线，无编号）
  - 更深的标题 → 加粗正文段落
- 例：文章只有 `##` / `###` / `####` 时，`##` 是一级、`###` 是二级；有 `###` / `#####` 时，`###` 是一级、`#####` 是二级。
- 已写 `01`、`02` 的一级标题会保留编号；没有编号时自动连续编号。
- 编号和标题属于同一个块，不能通过空行拆散。

## 正文与行内样式

- 普通段落会成为可按行跨页的正文。
- `**加粗**` 保留为粗体。
- 整段加粗且长度足够时，可自动推断为卡片。
- 只有明确以「高亮 / 划重点 / 卡片 / 注意 / 结论 / 金句 / 关键 / 判断 / 提醒 / 重点」等标签开头的部分加粗段落，才按卡片处理。

## 引用

```markdown
> 引用块适合放观点、说明和补充背景。
```

飞书原生引用导出后通常也是 `>`。引用结构优先于关键词，即使内容出现“金句”也不会变成卡片。

## 列表

```markdown
- 无序项目 A
- 无序项目 B

1. 有序项目 A
2. 有序项目 B
```

列表结构优先于内容关键词。列表项中出现“卡片”“重点”等词，仍保持列表。

## 表格

推荐使用标准 GFM pipe table：

```markdown
| 模式 | 适合 | 页数 |
| --- | --- | ---: |
| 有封面图 | 视觉强、产品感 | 6 |
| 无封面图 | 干货长文、教程 | 8 |
```

- 转换器通过“表头行 + `---` 分隔行”识别表格，不会把普通竖线文本误判为表格。
- 支持对齐冒号、首尾省略 `|`，以及单元格内用 `\|` 表示竖线。
- 飞书导出为 HTML `<table>` 时也会保留行列。
- 建议控制在 2–5 列；列太多应先精简，不要依赖横向滚动。

## 图片

```markdown
![图片说明](图片和附件/image%201.png)
```

- 推荐把 Markdown 和附件目录一起传入转换器。
- 支持中文、空格和 URL 编码路径。
- 连续竖图可能自动并排。

## 视频

视频不会进入小红书图文：

```markdown
演示视频.mov
```

转换器会跳过视频并生成提示。需要视频内容时，先截帧为图片，或在小红书发布页单独上传。

## 手动分页

单独一行的 `---`（也支持 `***`、`___`）表示**强制换页**。下一行内容会从新一页顶部开始；分页符本身不会出现在导出图里。

```markdown
## 03 这一页必须在这里结束

最后一段正文。

---

## 04 从这里起是新一页
```

注意：表格分隔行 `| --- | --- |` 不会被当成手动分页。不要用大量空行凑页码；高度仍由 Studio 按 3:4 版心自动测量，手动分页只是在自动分页之上加「硬断点」。
