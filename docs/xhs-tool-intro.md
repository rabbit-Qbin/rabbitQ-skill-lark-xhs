# 小兔Q彬 · 飞书 Markdown 转小红书图文

> **rabbitQ-skill-lark-xhs** — 飞书云文档导出的 Markdown 与全部附件，一键变成可编辑的小红书 3:4 图文 Studio。

---

## 这是什么？

写长文、做教程、发干货，最痛苦的不是写内容，而是**排版**。

小兔Q彬做的这个工具，专门解决一件事：

**把飞书云文档导出的 Markdown + 图片附件，自动分页成小红书 3:4 图文，并在浏览器里可视化编辑，最后批量导出 PNG。**

不用 PS，不用 Canva 一张张拼，写完就能发。

---

## 三步上手

1. **飞书云文档导出** → 得到 `.md` 文件 + 图片附件文件夹
2. **运行转换** → 生成 `xhs-studio.html` 本地 Studio
3. **浏览器编辑** → 调样式、改封面、拖块排序 → **批量导出 PNG ZIP**

```bash
node scripts/convert.js article.md
```

## 核心功能一览

| 功能 | 说明 |
|------|------|
| 自动分页 | 3:4 固定版心，按内容高度智能拆页 |
| 飞书附件解析 | 正文里的图片自动带入，路径自动修复 |
| 本地 Studio | 浏览器打开即可编辑，无需联网 |

## 样式自由组合

> 以下样式**全部可单独切换、自由混搭**，没有固定主题预设。

### 1. 图纸背景

给卡片加质感纹理，让图文不那么「素」。

- 支持多种纸张纹理
- 可与背景色、强调色叠加
- 选中「无」即纯色底

### 2. 背景主题色

控制卡片底色氛围：

- **米白** — 干净清爽，适合干货教程
- **暖白** — 柔和温暖，适合生活类内容
- **冷灰** — 偏专业感，适合工具测评

### 3. 强调色

控制高亮文字、下划线、卡片竖边、封面横杆等 accent 元素：

- **翠绿** — 默认品牌色，清新耐看
- **天蓝** — 科技感
- **珊瑚** — 活力感
- **琥珀** — 温暖感

### 4. 封面区

**两种模式：**

**有封面图（默认）**
- 上半页：封面图（可替换、可拖拽缩放）
- 下半页：标题 + 副标题 + 横杆装饰

**关闭封面图**
- 上半页：标题 + 副标题（样式与有图时一致）
- 下半页：正文自动接续，一屏看完开篇

**封面细节：**
- 标题自动缩放，长标题不会溢出
- 副标题最多 48 字、两行
- 副标题左侧竖条装饰（深绿色 accent）
- 粘贴标题自动净化外部样式

### 5. 卡片块（划重点 / 注意 / 金句）

选中文字 → 点「卡片」，自动生成带角标的强调卡片：

- **竖边** — 左侧粗色条，简洁有力
- **细框** — 四周边框，适合引用金句
- 角标自动推断：划重点 / 注意 / 金句 / 结论（≤3 字）

### 6. 引用块

选中**部分文字**（不是整段）→ 点「引用块」：

- 只转换选中内容，不会误变整段
- 左侧竖线 + 浅色底，适合放观点、金句

### 7. 序列列表

- Markdown 里的 `-` 和 `1.` 自动识别为列表
- 工具栏「序列」：多段选中 → 每条变一行
- 支持 Alt + 拖动调整顺序

### 8. 图片

- 单图 / 双图并排自动识别
- 点击可替换、删除
- 拖拽调整位置（Alt + 拖动）
- 竖图自动配对并排

### 9. 标题层级

- `##` 自动变成 01 / 02 编号标题
- 编号与标题文字垂直居中对齐

### 10. 文字样式

- **加粗** — 强调关键词
- **绿色字** — 高亮重点词
- **绿色下划线** — 底线高亮风格
- **划重点** — 一键变卡片块

---

## 编辑技巧

- **Alt + 拖动** — 移动卡片、引用块、序列、图片等任意块
- **重新分页** — 编辑后点「重新分页」自动重排
- **保存编辑** — 编辑状态写入 localStorage，刷新不丢
- **字体切换** — 微信体 / 宋体两种正文字体
