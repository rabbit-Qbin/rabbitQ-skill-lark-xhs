# rabbitQ 小红书 Studio 使用手册

`rabbitQ-skill-lark-xhs` 把飞书 Markdown 与附件转换为一个本地 HTML 编辑器。这里说明转换结果如何编辑，不重复安装步骤。

## 页面结构

- 顶部工具栏：文字、结构、图片、保存、复原和导出命令。
- 中间画布：固定 3:4 页面预览。
- 页码标签：切换封面和正文页。
- 右侧设置：封面、图纸、背景、强调色、图片和布局参数。

## 封面

### 显示封面图

- 上半页是封面图片框。
- 点击空图片框或双击现有图片即可替换。
- 拖动图片改变裁剪中心，滚轮改变缩放。
- 下半页可直接修改主标题和副标题。

### 关闭封面图

- 标题移动到上半页。
- 正文按原顺序接入下半页。
- 封面占位色设置自动隐藏。
- 再次显示封面图后，正文回到普通正文页，内容顺序不变。

## 文字工具

| 工具 | 行为 |
|---|---|
| B 加粗 | 正文默认 720；选中文字后点一次切为 700 不加粗，再点一次恢复 720 |
| H1 | 当前段落变成 01 / 02 编号标题 |
| H2 | 强调色字 + 2px 下划线（无编号） |
| 引用块 | 选中文字变成引用 |
| 有色字 | 使用当前强调色；再点一次取消 |
| 下划线 | 使用当前强调色的底部高亮；再点一次取消 |
| 卡片 | 生成带角标的卡片块 |
| `</>` 代码块 | 生成 macOS 三色圆点深色代码窗口；fenced code 会自动识别 |
| 无序列表图标 | 多段或多行转换为无序列表 |
| 有序列表图标 | 多段或多行转换为有序列表 |
| 块边界 `+ / −` | 每次增加或减少一行相邻空白；没有空行时隐藏 `−` |

正文页完全为空时会先保留一条真实可编辑行。正文末尾的空白区域按 58px 行网格响应：点击哪一行，Studio 才生成到该位置所需的真实空行并落下光标；拖动结构块或图片进入空白时，插入线也按同一网格吸附，松手后再写入空行。用户生成的这些定位行重新分页后仍保持 58px；普通跨页内容前的无意义页首空段仍会折叠，避免正文首字被压到第二行。

**一级标题 / 二级标题 / 引用块 / 卡片 / 代码块 / 序列** 共用同一套规则：光标在某块里再点**同一个**按钮 → 取消回正文；点**另一个**样式按钮 → 直接互切成新样式。代码块内部使用等宽字体并保留换行。

从普通正文新建代码块时，必须先选中同一个段落里的有效文字；跨段或整页选择会被拒绝，避免把整页内容误包进一个代码窗口。代码正文为 `32px`、行高 `1.55`；三色圆点保留原尺寸、颜色与间距，并在顶部工具栏内沿同一水平中心线排列。

在正文或列表中按 Enter 时，Studio 会保留光标前后全部文字和行内粗体；有序列表仍保持有序。自动重排会核对正文、图片、空行与分页符的完整顺序，异常时撤回并在顶部提示。

序列采用飞书式块交互：Enter 拆分当前项或续写下一项；空序列项再按 Enter、或在空项开头按 Backspace，会退出为普通空段；Shift+Enter 只在当前项内换行。非空序列项开头第一次按 Backspace 会去掉序号、退回正文并把光标留在正文开头；再按一次 Backspace 时，如果前方还是序列项，文字会并入前一项正文，不会重新生成一个序号。正文紧跟引用、卡片或代码块时，段首 Backspace 也会把文字并入前一块的正文位置。分页时序列可在完整项目之间连续跨页，但编号与该项正文保持绑定。块样式切换仍可处理整个连续序列；`Alt + 拖动` 只移动当前序列项。

分页只是普通正文连续流的 3:4 显示窗口，不是独立文本框。在下一页续写首字前按 Backspace，会继续删除上一页末尾字符；在引用、卡片或标题开头按 Enter，会在结构块前插入真实空段。标题、引用、卡片、代码块、图片和短表格是原子结构块，空间不足时整块换页；序列则在完整项目之间连续跨页。只有超出整页的长表格会按完整数据行拆分。结构块被推到下一页时，光标与新空段留在上一页末尾。

普通正文、序列、卡片与表格正文默认使用 `720` 加粗字重；选中文字点击 `B` 可切换为 `700` 不加粗，再点一次恢复 `720`。Markdown `**加粗**` 继续保留语义并使用 `720`。转换前运行 `node scripts/ensure-font.js`，确保所有电脑优先使用 `Noto Serif SC`；不依赖 macOS 私有字体映射。加粗不会自动附带浅蓝下划线；下划线只在原稿明确包含或手动点击 `U` 时出现。

## 表格

- 飞书导出的标准 Markdown pipe table 和 HTML `<table>` 会自动识别，不需要手动添加样式。
- 正式样式为：浅强调色表头、深强调色文字、表头上下 `1.5px` 实线、白底正文、行间虚线，最后一行保留 `2px` 浅色底线。
- 单元格文字可以直接编辑；悬浮后拖动左侧轻量六点手柄，或按住 `Alt` 拖动表格，可调整整张表的顺序。
- 短表保持完整；长表按完整数据行跨页并重复表头，不会把一行切成两半。
- 列数过多时应先精简列，否则 3:4 竖图中的单元格会过窄。

## 图片工具

1. 点击图片，出现蓝色选中边框。
2. 拖动图片本身调整裁剪位置。
3. 滚轮缩放图片内容。
4. 拖动右侧、下侧或右下控制点调整图片框。
5. 使用右侧按钮移动、并排或拆分图片块。
6. 按 Delete / Backspace 删除选中图片。

悬浮标题、卡片、引用、代码块、表格、单个序列项或图片后，可以拖动左侧小型六点手柄；也可以按住 `Alt` 直接拖动区块。拖动时只显示一条强调色插入横线，不生成透明区块克隆、气泡或整页遮罩，横线所在位置就是松手后的真实落点。总览编辑中可拖到另一页；图片落到目标页靠下且剩余高度不足时，插入线会自动上移到实际可放置位置，再把后续内容顺延。序列只移动当前项目；关闭封面图时也可拖到封面下半页。封面图本身不参与跨页拖动。

连续竖图初次转换时可能自动并排。用户手动拆分后，编辑器会保留用户选择。

## 主题

- **图纸背景**：无、方格纸、点阵纸、横线纸、蓝图格、细麻纸。
- **背景主题**：纯白（默认）、米白、薄荷、浅灰、煎黄、浅蓝、浅黄、浅粉、浅紫。
- **字体**：固定使用书刊宋体，不提供字体切换。
- **强调色**：知蓝（默认）、翠绿、莓粉、青缎、活力橙、雾紫。
- **封面占位色**：跟背景、跟强调、深色；只在显示封面图时出现。

强调色会统一影响有色字、下划线、封面装饰、卡片、序列标记和表格表头。

## 草稿、保存与复原

- 编辑时会自动写入当前浏览器的 localStorage。
- 同一篇源文件重新打开时会恢复草稿。
- 源文件指纹变化后，旧草稿不会静默覆盖新内容。
- localStorage 草稿和“保存编辑 HTML”都会做完整性检查；发现损坏时恢复安全检查点或源稿，并显示可见提示。
- “保存编辑 HTML”会生成一份嵌入当前状态的新 HTML，适合长期保存或发给别人继续编辑。
- “一键复原”会清除当前文章的本地草稿并恢复生成时状态，操作前会弹出确认。

## 导出

点击“批量导出 PNG ZIP”后：

- 每页导出为单独 PNG。
- 文件按 `01.png`、`02.png`……排序。
- 默认尺寸为 1080 × 1440。
- 内部先用 2 倍分辨率渲染，再高质量缩回 1080 × 1440，改善文字和细线边缘。
- 最后打包为 ZIP 下载。

导出前建议抽查封面、文字样式、引用、图片裁剪和最后一页，避免只看中间页。

## 视频说明

本工具生成小红书图文笔记，只处理图片。Markdown 中的视频链接或视频附件会被跳过，Studio 顶部会显示提示。需要视频时，请在小红书发布页单独上传，或先把视频截帧为图片。
